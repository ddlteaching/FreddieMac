# Introduction to Python {#IntroToPython}

## Base Python

Python works as a calculator


\begin{minted}[mathescape, fontsize=\small, xleftmargin=0.5em]{python}
1+1
\end{minted}
\begin{minted}[fontsize=\small, xleftmargin=0.5em, mathescape, frame = leftline]{text}
2
\end{minted}


\begin{minted}[mathescape, fontsize=\small, xleftmargin=0.5em]{python}
4*6
\end{minted}
\begin{minted}[fontsize=\small, xleftmargin=0.5em, mathescape, frame = leftline]{text}
24
\end{minted}


More generally, we can assign values to _variables_. For example:


\begin{minted}[mathescape, fontsize=\small, xleftmargin=0.5em]{python}
a = '123'
b = 123
c = 123.0
\end{minted}


Each of a, b and c are of different _types_:


\begin{minted}[mathescape, fontsize=\small, xleftmargin=0.5em]{python}
print(type(a))
print(type(b))
print(type(c))
\end{minted}
\begin{minted}[fontsize=\small, xleftmargin=0.5em, mathescape, frame = leftline]{text}
<class 'str'>
<class 'int'>
<class 'float'>
\end{minted}
