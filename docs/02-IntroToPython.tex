\subsection{Base Python}\label{base-python}

Python works as a calculator


\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{l+m+mi}{1}\PY{o}{+}\PY{l+m+mi}{1}
\end{Verbatim}

\begin{Verbatim}[commandchars=\\\{\},frame=leftline,fontsize=\small, xleftmargin=0.5em]
2
\end{Verbatim}




\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{l+m+mi}{4}\PY{o}{*}\PY{l+m+mi}{6}
\end{Verbatim}

\begin{Verbatim}[commandchars=\\\{\},frame=leftline,fontsize=\small, xleftmargin=0.5em]
24
\end{Verbatim}

More generally, we can assign values to \emph{variables}. For example:


\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{n}{a} \PY{o}{=} \PY{l+s+s1}{\PYZsq{}}\PY{l+s+s1}{123}\PY{l+s+s1}{\PYZsq{}}
\PY{n}{b} \PY{o}{=} \PY{l+m+mi}{123}
\PY{n}{c} \PY{o}{=} \PY{l+m+mf}{123.0}
\end{Verbatim}

Python has several \emph{types} of variables, but at a basic level, we
will be using string or character variables (\texttt{str}), and
numerical variables (\texttt{float} or real numbers, and \texttt{int} or
integers). For example, each of a, b and c are of different
\emph{types}:


\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{k}{print}\PY{p}{(}\PY{n+nb}{type}\PY{p}{(}\PY{n}{a}\PY{p}{)}\PY{p}{)}
\PY{k}{print}\PY{p}{(}\PY{n+nb}{type}\PY{p}{(}\PY{n}{b}\PY{p}{)}\PY{p}{)}
\PY{k}{print}\PY{p}{(}\PY{n+nb}{type}\PY{p}{(}\PY{n}{c}\PY{p}{)}\PY{p}{)}
\end{Verbatim}

\begin{Verbatim}[commandchars=\\\{\},frame=leftline,fontsize=\small, xleftmargin=0.5em]
<class 'str'>
<class 'int'>
<class 'float'>
\end{Verbatim}

Even though these three values will appear the same in a spreadsheet
like Excel, they are distinct to a computer. In fact, if we test if
\emph{a} and \emph{b} are the same:


\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{n}{a} \PY{o}{==} \PY{n}{b}
\end{Verbatim}

\begin{Verbatim}[commandchars=\\\{\},frame=leftline,fontsize=\small, xleftmargin=0.5em]
False
\end{Verbatim}

However, Python is smart about numbers, so,


\begin{Verbatim}[commandchars=\\\{\},frame=single,fontsize=\small, xleftmargin=0.5em]
\PY{n}{b} \PY{o}{==} \PY{n}{c}
\end{Verbatim}

\begin{Verbatim}[commandchars=\\\{\},frame=leftline,fontsize=\small, xleftmargin=0.5em]
True
\end{Verbatim}
